# 説明
```sh
cargo compete new abc320
cargo member include abc320
```

コンテストのディレクトリ内で以下のコマンド. テストしてから提出。
```sh
cargo compete s a
```

```sh
cargo compete s a --no-test
```

```sh
cargo compete open
```

```sh
cargo compete test
```


普通に実行したいときは、以下
```sh
cargo run --bin abc320-a
```

# 初期セットアップ
```sh
cargo install cargo-compete
cargo install cargo-member
```
